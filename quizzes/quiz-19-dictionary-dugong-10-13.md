# Week 19 Quiz — Dictionary of the Dugong — Ages 10–13

> *10 code-focused, CBE-aligned questions for Week 19: Dictionary of the Dugong.*

## Questions

1. What is a dictionary?
2. Write a dictionary that stores a dugong's name and length.
3. How do you get the value for a key?
4. What happens if you ask for a key that is not in the dictionary?
5. How do you add a new key to a dictionary?
6. Which data type would you use for the key?
7. Give one real-world use of a dictionary for marine life.
8. Write code to print the length of a dugong from a dictionary.
9. What is the difference between a list and a dictionary?
10. Which CBE strand uses dictionaries?

## Answer Key

1. A collection of key-value pairs.
2. dugong = {'name': 'Dottie', 'length': 3}
3. dictionary['key']
4. You get an error unless you use .get().
5. dictionary['new_key'] = value
6. Usually a string.
7. Store species names with their conservation status.
8. print(dugong['length'])
9. A list uses numbers; a dictionary uses keys.
10. Data.
