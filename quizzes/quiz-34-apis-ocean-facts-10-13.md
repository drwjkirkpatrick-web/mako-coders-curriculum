# Week 34 Quiz — APIs — Fetching Ocean Facts — Ages 10–13

> *10 code-focused, CBE-aligned questions for Week 34: APIs — Fetching Ocean Facts.*

## Questions

1. What is an API?
2. What format do many APIs use?
3. What does JSON look like?
4. Write a request to get an ocean fact API in words.
5. Why would a program use an API instead of a webpage?
6. What is a key in JSON?
7. What is a value in JSON?
8. Name one kind of data an ocean API might give.
9. What happens if the API does not respond?
10. Which CBE strand links to APIs?

## Answer Key

1. A way for programs to ask for data from a server.
2. JSON.
3. Data in curly braces with keys and values.
4. Send a GET request to the API URL and read the response.
5. An API returns clean data, not a whole page.
6. The name before a colon.
7. The data after a colon.
8. Species facts, tides, weather.
9. Your program should show a friendly error.
10. Networks / Data.
