# Week 16 Quiz — Lists of Kenyan Marine Life — Ages 14–17

> *10 code-focused, CBE-aligned questions for Week 16: Lists of Kenyan Marine Life.*

## Questions

1. Write a Python list that holds 5 real Kenyan ocean species.
2. What is list indexing and slicing?
3. What does species[2:4] return for a 5-item list?
4. How do you insert an item at the start of a list?
5. Write code to remove the last item from a list and store it.
6. What is the difference between append() and extend()?
7. How do you check if 'mako' is in a list?
8. Explain list mutation with an example.
9. Write a loop that prints every species in a list.
10. Map lists to a CBE Data strand outcome.

## Answer Key

1. species = ['spinner_dolphin', 'green_turtle', 'whale_shark', 'dugong', 'humpback_dolphin']
2. Indexing gets one item; slicing gets a range of items.
3. Items at index 2 and 3.
4. species.insert(0, 'new_item')
5. last = species.pop()
6. append() adds one item; extend() adds multiple items.
7. 'mako' in species
8. species[0] = 'mako' changes the first item.
9. for s in species:
    print(s)
10. Learners store and manipulate ordered collections of data.
