# Week 08 Quiz — Whale Shark Filter — Variables — Ages 10–13

> *10 code-focused, CBE-aligned questions for Week 08: Whale Shark Filter — Variables.*

## Questions

1. What is a variable?
2. Which Scratch block makes a variable?
3. Write the blocks to set 'score' to 0 when the flag is clicked.
4. What happens when you use 'change score by 1'?
5. Name three things you could store in a variable for a whale shark game.
6. Why is a variable better than writing the number 100 directly in many places?
7. What block shows a variable's value on the stage?
8. If 'plankton' starts at 20 and you 'change plankton by -3', what is the new value?
9. Write a script that counts how many times a sprite is clicked.
10. What type of value does a score variable usually hold?

## Answer Key

1. A box that holds a value that can change.
2. Make a Variable
3. When green flag clicked → set score to 0.
4. The score variable increases by 1.
5. Score, health, speed, level.
6. You can change it in one place.
7. show variable [score]
8. 17.
9. When sprite clicked → change clicks by 1.
10. A number.
