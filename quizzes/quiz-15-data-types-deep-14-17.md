# Week 15 Quiz — Data Types of the Deep — Ages 14–17

> *10 code-focused, CBE-aligned questions for Week 15: Data Types of the Deep.*

## Questions

1. Name Python's three basic data types and give an example of each.
2. What is type checking and why is it useful?
3. What is the output of type(12.0) in Python?
4. Write code that safely converts user input to an integer.
5. What is the difference between '5' and 5 in Python?
6. Explain why mixing types in arithmetic can cause bugs.
7. What data type would you use for a GPS coordinate like -4.05?
8. Write a Python expression that checks if a species string starts with 'whale'.
9. How would you store a true/false flag for 'tagged'?
10. Map data types to a CBE Data strand outcome.

## Answer Key

1. int (5), float (5.5), str ('shark'), bool (True)
2. Verifying the kind of value a variable holds to prevent errors.
3. <class 'float'>
4. age = int(input('Enter age: '))
5. '5' is a string; 5 is an integer.
6. Adding a string to a number raises a TypeError.
7. float
8. species.startswith('whale')
9. tagged = False
10. Learners select appropriate data types for real-world measurements.
