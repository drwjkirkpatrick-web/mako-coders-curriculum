# Week 03 Quiz — Hatchling Dash — Sprites & Motion — Ages 10–13

> *10 code-focused, CBE-aligned questions for Week 03: Hatchling Dash — Sprites & Motion.*

## Questions

1. Which block changes a sprite's position to a specific x and y coordinate?
2. What does the 'glide' block do?
3. If a sprite is at x=0, y=0 and you use 'change x by 50', where does it go?
4. Which block makes a sprite point towards the mouse pointer?
5. Write the blocks to make a turtle sprite move in a circle.
6. What is a costume in Scratch?
7. How do you make a sprite flip to its next costume?
8. Which Motion block reports where the sprite is?
9. What does 'set rotation style left-right' stop?
10. Create a 5-block script that makes a hatchling turtle walk to the sea and switch costume.

## Answer Key

1. go to x: _ y: _
2. Moves a sprite smoothly to a point over time.
3. To x=50, y=0.
4. point towards mouse-pointer
5. Repeat 12: move 10 steps, turn 30 degrees.
6. A different picture for the same sprite.
7. Use the 'next costume' block.
8. x position / y position
9. It stops the sprite from rotating upside down.
10. When green flag clicked → go to x:-200 y:-100 → repeat 10: move 10 steps, next costume.
