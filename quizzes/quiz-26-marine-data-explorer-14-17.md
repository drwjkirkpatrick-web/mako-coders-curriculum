# Week 26 Quiz — Marine Data Explorer Mini-Project — Ages 14–17

> *10 code-focused, CBE-aligned questions for Week 26: Marine Data Explorer Mini-Project.*

## Questions

1. Write a short design brief for a Marine Data Explorer app.
2. How would you structure the project files?
3. What is a minimum viable product?
4. Write pseudocode for loading data, filtering it, and plotting a chart.
5. How would you handle bad or missing CSV values?
6. What makes a chart in a project trustworthy?
7. Explain how to add a user menu in a text-based program.
8. Which testing strategy proves your data pipeline works?
9. How would you document the project for another coder?
10. Map the mini-project to CBE competencies.

## Answer Key

1. The app lets users view, filter, and chart Kenyan marine species counts from a CSV file.
2. main.py, data.csv, charts/, README.md
3. The simplest version that works end-to-end.
4. load CSV; filter rows; calculate counts; plot bar chart; show/save.
5. Skip rows, use defaults, or report errors.
6. Clear labels, source citation, and correct data.
7. Print options, accept input, use if/elif to call functions.
8. Test with a small known CSV and compare output.
9. README with setup, file list, and function descriptions.
10. Data handling, Creativity, Critical Thinking, Communication.
