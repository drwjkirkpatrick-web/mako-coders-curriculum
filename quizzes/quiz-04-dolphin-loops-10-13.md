# Week 04 Quiz — Dolphin Loops — Ages 10–13

> *10 code-focused, CBE-aligned questions for Week 04: Dolphin Loops.*

## Questions

1. Which block repeats a set of actions a specific number of times?
2. How many times will a spinner dolphin spin if the repeat block says 8?
3. What is the difference between 'repeat' and 'forever'?
4. Write a loop that makes a dolphin turn 360 degrees in 10 equal steps.
5. Why are loops useful in coding?
6. If you have 'repeat 5' around a 'move 10 steps' block, how far does the sprite move?
7. What happens if a loop has no blocks inside it?
8. Which block category contains the repeat loops in Scratch?
9. Write a script that makes a dolphin spin forever when the flag is clicked.
10. How do you stop a 'forever' loop in Scratch?

## Answer Key

1. repeat 10
2. 8 times.
3. Repeat stops after a set number; forever never stops.
4. Repeat 10: turn 36 degrees.
5. They let you repeat code without writing it many times.
6. 50 steps.
7. Nothing repeats.
8. Control.
9. When green flag clicked → forever: turn 15 degrees.
10. Click the red stop sign.
