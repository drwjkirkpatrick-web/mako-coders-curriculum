# Lesson 04: Dolphin Loops

- **Module Reference:** Prompt 04
- **Duration:** 2 hours (10-minute break after guided practice)
- **Age Group:** CBE Junior School (12–15 years)
- **Coding Tool:** Scratch 3

## CBE Strand Mapping

| CBE Learning Area | Strand | Sub-strand |
|-------------------|--------|------------|
| Computer Studies / ICT | Algorithms and Programming | Loops and Repetition |

## Learning Outcomes

By the end of the lesson the learner should be able to:
1. **Explain** why loops save time when repeating instructions.
2. **Apply** a `repeat` block to animate a dolphin spin.
3. **Value** using fewer blocks to create the same effect.

## Key Inquiry Question

*How can a spinner dolphin teach us to repeat actions without writing the same code again and again?*

## Resources

- Computers or tablets with Scratch 3 (offline editor preferred)
- Projector for live demo
- Printed dolphin sprite cards (optional)
- Starter project `starter-dolphin-loop.sb3` (optional)

## Local Context: Spinner Dolphin

- **Common name:** Spinner dolphin
- **Scientific name:** *Stenella longirostris*
- **Swahili name:** Pomboo
- **Habitat:** Pelagic and coastal waters of the Western Indian Ocean, often near Kenya’s reef drop-offs
- **Why this species:** Spinner dolphins are famous for acrobatic spins — a perfect metaphor for rotation and repetition.
- **Conservation note:** Boat noise and harassment can disrupt resting pods. Learners should learn to "code respectfully" and share this message.

## Lesson Development

### Intro hook (10 min)
Show a 30-second video or image of a spinner dolphin pod. Ask: "If we wanted every dolphin to spin 10 times, would we write 'turn, turn, turn...' 10 times?"

### Step 1 — Concept + teacher demo (20 min)
Explain the `repeat` block. Live-code a single dolphin that turns 36° ten times to make a full 360° spin. Use the Swahili word *pomboo*.

### Step 2 — Guided practice (30 min)
Learners open the starter project and:
- Change the number of repeats.
- Change the turn angle.
- Add a second dolphin that spins in the opposite direction.

### Break (10 min)

### Step 3 — Independent practice (35 min)
Create a pod of three dolphins that spin together when the green flag is clicked. Checklist:
- [ ] At least three dolphin sprites
- [ ] Each sprite uses a repeat block
- [ ] Green flag starts all spins
- [ ] A conservation message appears as a say-bubble

### Step 4 — Sharing and feedback (15 min)
Two learners demo. Peers give one glow and one grow using the CBE rubric language.

## Assessment

| Criterion | BE (0–39%) | AE (40–59%) | ME (60–79%) | EE (80–100%) |
|-----------|------------|-------------|---------------|----------------|
| Uses a loop correctly | Needs help to place a repeat block | Places repeat block but number is wrong | Places repeat block and correct number | Uses nested or conditional loop |
| Explains the loop concept | Cannot explain in own words | Explains with teacher prompting | Explains clearly to a peer | Explains and gives an example |
| Adds a conservation message | No message | Message copied from teacher | Original short message | Original message + explains why it matters |

## Extended Activity

Make the dolphins spin faster when the space key is pressed. Research one more fact about spinner dolphins and add it to the project.

## Extra Credit Challenge

**Nested pattern challenge.** Use one loop inside another to draw a coral reef tile or turtle-shell pattern with at least 6 shapes.

## Homework Assignment

Count how many times you repeat an action at home today (brushing teeth, climbing stairs). Write the number and one way a loop could model it.
