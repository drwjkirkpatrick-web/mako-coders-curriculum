# Lesson Plan Template

## File name pattern

`lessons/lesson-NN-slug.md`

## Required sections

1. Title + Module Reference + Duration + Age Group + Coding Tool
2. CBE Strand Mapping (table)
3. 3 Learning Outcomes (Bloom's verbs: Explain, Apply, Value)
4. Key Inquiry Question
5. Resources
6. Local Context: species name, Latin name, Swahili/local name, habitat, why this species, conservation note
7. Lesson Development: Intro hook + Step 1 (concept + code) + Step 2 (guided practice + code) + Step 3 (independent practice + checklist) + Step 4 (sharing/feedback)
8. Assessment table (BE/AE/ME/EE, 3+ criteria rows)
9. Extended Activity
10. Extra Credit Challenge
11. Homework Assignment
12. Weekly Quiz (links to `quizzes/quiz-NN-ages-10-13.md` and `quizzes/quiz-NN-ages-14-17.md`)
13. Differentiation table (support vs advanced)
14. Teacher Reflection checklist
15. RPF & CBE Cross-Reference table
