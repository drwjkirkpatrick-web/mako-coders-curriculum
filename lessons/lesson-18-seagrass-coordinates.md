# Lesson 18: Seagrass Coordinates

- **Module Reference:** Prompt 18
- **Duration:** 2 hours (10-minute break after guided practice)
- **Age Group:** CBE Junior/Senior School (12-15 years)
- **Coding Tool:** Python + Turtle

## CBE Strand Mapping

| CBE Learning Area | Strand | Sub-strand |
|-------------------|--------|------------|
| Computer Studies / ICT | Data | Data and Information |

## Learning Outcomes

By the end of the lesson the learner should be able to:
1. **Explain** explain how data is stored and structured.
2. **Apply** apply data structures to model marine information.
3. **Value** value data as a tool for conservation.

## Key Inquiry Question

*How can the Seagrass beds (mwani wa baharini) help us learn about nested lists and coordinates?*

## Resources

- Python + Turtle installed on learner devices (offline setup where possible)
- Projector for live demo
- Starter files or handouts as needed
- Reference material from `research/domain-reference.md`

## Local Context: Seagrass beds

- **Common name:** Seagrass beds
- **Swahili name:** mwani wa baharini
- **Habitat:** shallow coastal seabed; nursery for fish and food for turtles/dugongs
- **Why this species:** This species anchors the week's concept because its behaviour or status connects naturally to nested lists and coordinates.
- **Conservation note:** threatened by salt pans, shrimp farms, sand mining, and siltation.

## Lesson Development

### Intro hook (10 min)
Share a quick fact about the Seagrass beds (mwani wa baharini). Ask a question that links the species to the coding concept: "How can we use code to seagrass...?"

### Step 1 — Concept + teacher demo (20 min)
Teach the week's concept (nested lists and coordinates) using a short live demo in Python + Turtle. Connect every block or line of code back to the Seagrass beds.

### Step 2 — Guided practice (30 min)
Learners follow along with a starter activity. They make small changes and test frequently. The teacher circulates and uses the anchor species as a memory hook.

### Break (10 min)

### Step 3 — Independent practice (35 min)
Create the main deliverable: **seagrass grid map**. Checklist:
- [ ] Core concept from the lesson is present
- [ ] Anchor species (Seagrass beds) is referenced in the project
- [ ] Code runs without critical errors
- [ ] One conservation message or note is included

### Step 4 — Sharing and feedback (15 min)
Two or three learners demo their work. Peers give one "glow" (what worked well) and one "grow" (one idea to improve) using CBE rubric language.
