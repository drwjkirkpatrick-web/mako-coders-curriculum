- **Module Reference:** Prompt 44
- **Duration:** 2 hours (10-minute break after guided practice)
- **Age Group:** CBE Junior/Senior School (15-18 years)
- **Coding Tool:** Python + Pygame Zero

## Learning Outcomes

By the end of the lesson the learner should be able to:
1. **Explain** explain the sequence of instructions.
2. **Apply** apply control structures to solve a problem.
3. **Value** value clear, ordered thinking in code.

## Key Inquiry Question

*How can the Shortfin mako shark (papa mako) help us learn about game loop and keyboard events?*

## Resources

- Python + Pygame Zero installed on learner devices (offline setup where possible)

- Starter files or handouts as needed

## Local Context: Shortfin mako shark

- **Common name:** Shortfin mako shark
- **Swahili name:** papa mako
- **Habitat:** tropical and temperate Indian Ocean waters; juveniles near coast, adults offshore
- **Why this species:** This species anchors the week's concept because its behaviour or status connects naturally to game loop and keyboard events.
- **Conservation note:** vulnerable globally; often taken as bycatch and targeted catch.

## Lesson Development

### Intro hook (10 min)
Share a quick fact about the Shortfin mako shark (papa mako). Ask a question that links the species to the coding concept: "How can we use code to keyboard-controlled...?"

### Step 1 — Concept + teacher demo (20 min)
Teach the week's concept (game loop and keyboard events) using a short live demo in Python + Pygame Zero. Connect every block or line of code back to the Shortfin mako shark.

### Step 2 — Guided practice (30 min)
Learners follow along with a starter activity. They make small changes and test frequently. The teacher circulates and uses the anchor species as a memory hook.

### Break (10 min)

### Step 3 — Independent practice (35 min)
Create the main deliverable: **keyboard-controlled mako**. Checklist:
- [ ] Core concept from the lesson is present
- [ ] Anchor species (Shortfin mako shark) is referenced in the project
- [ ] Code runs without critical errors
- [ ] One conservation message or note is included

### Step 4 — Sharing and feedback (15 min)
Two or three learners demo their work. Peers give one "glow" (what worked well) and one "grow" (one idea to improve) using CBE rubric language.

## Extended Activity

Research one additional fact about the Shortfin mako shark and add it to the project. Alternatively, modify the code to respond to a second input or display extra data.

## Extra Credit Challenge

**Nested pattern challenge.** Use one loop inside another to draw a coral reef tile or turtle-shell pattern with at least 6 shapes.

## Homework Assignment

Count how many times you repeat an action at home today (brushing teeth, climbing stairs). Write the number and one way a loop could model it.
