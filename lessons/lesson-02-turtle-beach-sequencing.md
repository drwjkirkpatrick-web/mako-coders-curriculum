- **Module Reference:** Prompt 02
- **Duration:** 2 hours (10-minute break after guided practice)
- **Age Group:** CBE Junior/Senior School (12-15 years)
- **Coding Tool:** Scratch 3

## Learning Outcomes

By the end of the lesson the learner should be able to:
1. **Explain** explain the sequence of instructions.
2. **Apply** apply control structures to solve a problem.
3. **Value** value clear, ordered thinking in code.

## Key Inquiry Question

*How can the Green sea turtle (kobe ya kijani) help us learn about sequencing instructions in order?*

## Resources

- Scratch 3 installed on learner devices (offline setup where possible)
- Projector for live demo
- Starter files or handouts as needed

## Local Context: Green sea turtle

- **Common name:** Green sea turtle
- **Swahili name:** kobe ya kijani
- **Habitat:** seagrass beds and nesting beaches along the Kenyan coast
- **Why this species:** This species anchors the week's concept because its behaviour or status connects naturally to sequencing instructions in order.
- **Conservation note:** endangered; nesting beaches need darkness, quiet, and protection from plastic.

## Lesson Development

### Intro hook (10 min)
Share a quick fact about the Green sea turtle (kobe ya kijani). Ask a question that links the species to the coding concept: "How can we use code to ordered...?"

### Step 1 — Concept + teacher demo (20 min)
Teach the week's concept (sequencing instructions in order) using a short live demo in Scratch 3. Connect every block or line of code back to the Green sea turtle.

### Step 2 — Guided practice (30 min)
Learners follow along with a starter activity. They make small changes and test frequently. The teacher circulates and uses the anchor species as a memory hook.

### Break (10 min)

### Step 3 — Independent practice (35 min)
Create the main deliverable: **ordered movement script**. Checklist:
- [ ] Core concept from the lesson is present
- [ ] Anchor species (Green sea turtle) is referenced in the project
- [ ] Code runs without critical errors
- [ ] One conservation message or note is included

### Step 4 — Sharing and feedback (15 min)
Two or three learners demo their work. Peers give one "glow" (what worked well) and one "grow" (one idea to improve) using CBE rubric language.

## Extended Activity

Research one additional fact about the Green sea turtle and add it to the project. Alternatively, modify the code to respond to a second input or display extra data.

## Extra Credit Challenge

**Design a second scene.** Create a short Scratch animation or Python sketch that shows the Green sea turtle at a different time of day (dawn, noon, or night). Add at least one new sprite or shape and explain in one sentence why you chose that time.

## Homework Assignment

Spend 15–20 minutes at home exploring the Green sea turtle online or in a book. Write down one new fact and one question you would like to ask Mako next week. Bring your fact and question to class.
