- **Module Reference:** Prompt 06
- **Duration:** 2 hours (10-minute break after guided practice)
- **Age Group:** CBE Junior/Senior School (12-15 years)
- **Coding Tool:** Scratch 3

## Learning Outcomes

By the end of the lesson the learner should be able to:
1. **Explain** explain the sequence of instructions.
2. **Apply** apply control structures to solve a problem.
3. **Value** value clear, ordered thinking in code.

## Key Inquiry Question

*How can the Indian Ocean humpback dolphin (pomboo ya Bahari Hindi) help us learn about if/else conditional decisions?*

## Local Context: Indian Ocean humpback dolphin

- **Common name:** Indian Ocean humpback dolphin
- **Swahili name:** pomboo ya Bahari Hindi
- **Habitat:** shallow coastal waters, river mouths, and mangrove edges
- **Why this species:** This species anchors the week's concept because its behaviour or status connects naturally to if/else conditional decisions.
- **Conservation note:** critically endangered in East Africa; threatened by gillnets and coastal development.

## Lesson Development

### Intro hook (10 min)
Share a quick fact about the Indian Ocean humpback dolphin (pomboo ya Bahari Hindi). Ask a question that links the species to the coding concept: "How can we use code to species...?"

### Step 1 — Concept + teacher demo (20 min)
Teach the week's concept (if/else conditional decisions) using a short live demo in Scratch 3. Connect every block or line of code back to the Indian Ocean humpback dolphin.

### Step 2 — Guided practice (30 min)
Learners follow along with a starter activity. They make small changes and test frequently. The teacher circulates and uses the anchor species as a memory hook.

### Break (10 min)

### Step 3 — Independent practice (35 min)
Create the main deliverable: **species ID sprite**. Checklist:
- [ ] Core concept from the lesson is present
- [ ] Anchor species (Indian Ocean humpback dolphin) is referenced in the project
- [ ] Code runs without critical errors
- [ ] One conservation message or note is included

### Step 4 — Sharing and feedback (15 min)
Two or three learners demo their work. Peers give one "glow" (what worked well) and one "grow" (one idea to improve) using CBE rubric language.

## Extended Activity

Research one additional fact about the Indian Ocean humpback dolphin and add it to the project. Alternatively, modify the code to respond to a second input or display extra data.

## Extra Credit Challenge

**Multi-species identifier.** Add one more species to your conditional decision tree and test it with three different inputs.

## Homework Assignment

List three if/then rules you follow at home or school. Convert one of them into pseudocode or Scratch blocks.
