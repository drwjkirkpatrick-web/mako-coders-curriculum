# Lesson 23: Plot the Coast

- **Module Reference:** Prompt 23
- **Duration:** 2 hours (10-minute break after guided practice)
- **Age Group:** CBE Junior/Senior School (12-15 years)
- **Coding Tool:** Python + Turtle

## CBE Strand Mapping

| CBE Learning Area | Strand | Sub-strand |
|-------------------|--------|------------|
| Computer Studies / ICT | Data | Data and Information |

## Learning Outcomes

By the end of the lesson the learner should be able to:
1. **Explain** explain how data is stored and structured.
2. **Apply** apply data structures to model marine information.
3. **Value** value data as a tool for conservation.

## Key Inquiry Question

*How can the Survey results (matokeo ya utafiti) help us learn about drawing simple charts?*

## Resources

- Python + Turtle installed on learner devices (offline setup where possible)
- Projector for live demo
- Starter files or handouts as needed
- Reference material from `research/domain-reference.md`

## Local Context: Survey results

- **Common name:** Survey results
- **Swahili name:** matokeo ya utafiti
- **Habitat:** data from the 2023 aerial megafauna survey along Kenya's coast
- **Why this species:** This species anchors the week's concept because its behaviour or status connects naturally to drawing simple charts.
- **Conservation note:** survey data guides conservation action and marine protected area planning.

## Lesson Development

### Intro hook (10 min)
Share a quick fact about the Survey results (matokeo ya utafiti). Ask a question that links the species to the coding concept: "How can we use code to coastal...?"

### Step 1 — Concept + teacher demo (20 min)
Teach the week's concept (drawing simple charts) using a short live demo in Python + Turtle. Connect every block or line of code back to the Survey results.

### Step 2 — Guided practice (30 min)
Learners follow along with a starter activity. They make small changes and test frequently. The teacher circulates and uses the anchor species as a memory hook.

### Break (10 min)

### Step 3 — Independent practice (35 min)
Create the main deliverable: **coastal bar chart**. Checklist:
- [ ] Core concept from the lesson is present
- [ ] Anchor species (Survey results) is referenced in the project
- [ ] Code runs without critical errors
- [ ] One conservation message or note is included

### Step 4 — Sharing and feedback (15 min)
Two or three learners demo their work. Peers give one "glow" (what worked well) and one "grow" (one idea to improve) using CBE rubric language.

## Extended Activity

Research one additional fact about the Survey results and add it to the project. Alternatively, modify the code to respond to a second input or display extra data.

## Extra Credit Challenge

**Collect your own data.** Record three real or realistic observations about marine life near your community (or from a video). Add them to your project data file and explain any trends you notice.

## Homework Assignment

Find one real news article or report about Kenyan marine conservation. Summarise it in 3–5 sentences and note any numbers or data mentioned.
