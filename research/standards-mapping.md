# Mako Coders — Kenya CBE Standards Mapping

> *Curriculum alignment document for the Mako Coders Kenyan ocean-themed coding programme.*
> *Research completed: 3 July 2026 from KICD CBE framework, KNEC assessment guidance, and Kenya Marine Mammal Network / IFAW marine-life data.*

---

## 1. Kenya CBE Context

Kenya replaced the 8-4-4 system with **CBC (Competency-Based Curriculum)** in 2017. In 2025 the Ministry of Education officially rebranded the reform as **CBE (Competency-Based Education)** — "Elimu Thabiti" — to signal that the reform now covers the whole education ecosystem (curriculum, pedagogy, resources, assessment), not just the syllabus.

For practical classroom use, the elements that matter for Mako Coders are unchanged:

| Element | What it means for this coding course |
|--------|--------------------------------------|
| **2-6-3-3 structure** | Learners targeted are mainly CBE Junior School (Grade 7–9, ages 12–15) and Senior School STEM pathway (Grade 10+, ages 15–18). |
| **Strand / Sub-strand** | Every module maps to a CBE strand (e.g. *Algorithms and Programming*) and sub-strand (e.g. *Control Structures*). |
| **7 core competencies** | Each lesson explicitly develops Communication, Self-efficacy, Critical Thinking, Creativity, Citizenship, Digital Literacy, and Learning to Learn. |
| **Rubric bands** | Assessment uses BE / AE / ME / EE (0–39%, 40–59%, 60–79%, 80–100%). |
| **40% school-based / 60% national** | Coursework, portfolios, and projects carry significant weight. |
| **ICT compulsory** | ICT is now compulsory for all Senior School learners; Computer Studies sits in the STEM → Applied Sciences cluster. |

## 2. CBE 7 Core Competencies

1. **Communication & Collaboration** — learners explain code, pair-debug, and present projects.
2. **Self-efficacy** — learners tackle harder tasks after small wins; ocean metaphors build confidence.
3. **Critical Thinking & Problem Solving** — debugging, decomposition, and algorithm design.
4. **Creativity & Imagination** — art, storytelling, and game design using marine themes.
5. **Citizenship** — conservation, marine stewardship, and ethical technology use.
6. **Digital Literacy** — safe, responsible, and effective use of computers and the internet.
7. **Learning to Learn** — reflection, note-taking, and using help resources.

## 3. Assessment Rubric (CBE Four-Level Bands)

| Band | Label | Range | Learner evidence in Mako Coders |
|------|-------|-------|---------------------------------|
| **BE** | Below Expectations | 0–39% | Cannot run starter code independently; needs step-by-step help for every edit. |
| **AE** | Approaching Expectations | 40–59% | Runs starter code and makes 1–2 guided changes; explains what changed in simple terms. |
| **ME** | Meeting Expectations | 60–79% | Completes the main task, tests it, fixes 1 bug, and explains the concept to a peer. |
| **EE** | Exceeding Expectations | 80–100% | Extends the project with an original feature and helps another learner debug theirs. |

## 4. Computer Studies / ICT Strand Alignment

The 52-week plan maps to the CBE Computer Studies / ICT strands that are most relevant for coding:

| CBE Strand | Sub-strands touched in Mako Coders | Weeks |
|------------|------------------------------------|-------|
| **Algorithms & Programming** | Sequencing, variables, loops, conditionals, functions, events, debugging, decomposition | 1–20, 35–44 |
| **Data & Information** | Data types, lists/arrays, dictionaries, data collection, visualisation | 21–26, 43 |
| **Computers & Hardware** | Input/output, micro:bit/RPi Pico, sensors, actuators | 27–32 |
| **Networks & Communications** | Internet basics, safe sharing, APIs, web hooks | 33–34 |
| **Cybersecurity & Ethics** | Passwords, privacy, digital footprint, conservation ethics | 41–42 |
| **Project Development** | Design → Code → Create → Connect → Evaluate cycle | 45–52 |

## 5. Mako Coders Course Overview

- **Target group:** CBE Junior School (Grade 7–9) and Senior School STEM beginners.
- **Format:** Once per week, 2 hours per session, with one 10-minute break.
- **Language:** English with Kiswahili key terms and marine-life local names where available.
- **Tool progression:** Block-based (Scratch 3 / micro:bit MakeCode) → text-based (Python + Turtle / Pygame Zero) → project integration.
- **Final project:** A playable, conservation-themed **Mako Shark Research Dashboard / Game** that uses real species data from the Kenyan coast.

## 6. Kenyan Marine Life Context

Research from IFAW, Kenya Wildlife Service, Kenya Marine Mammal Network, and WCS confirms the following real species off the Kenyan coast:

| Species | Scientific name | Local / Swahili name | Status | Coding metaphor |
|---------|-----------------|----------------------|--------|-----------------|
| **Shortfin mako shark** | *Isurus oxyrinchus* | Papa (shark) | Vulnerable globally; present in Indian Ocean; heavily fished | Fast, efficient hunter → optimisation, real-time data, final project mascot |
| **Whale shark** | *Rhincodon typus* | Papa mafia | Endangered | Gentle filter-feeder → big-data filtering, loops through lists |
| **Spinner dolphin** | *Stenella longirostris* | Pomboo | Common in WIO; threatened by tourism pressure | Spinning pods → rotation/animation, angles in Scratch/Turtle |
| **Indian Ocean humpback dolphin** | *Sousa plumbea* | Pomboo ya Bahari Hindi | Critically endangered in East Africa | Rare sightings → conditional probability, if/else |
| **Green sea turtle** | *Chelonia mydas* | Kobe ya kijani | Endangered | Nesting cycles → loops, timers, state machines |
| **Hawksbill sea turtle** | *Eretmochelys imbricata* | Kobe ya pwani | Critically endangered | Patterned shell → pattern drawing, nested loops |
| **Dugong** | *Dugong dugon* | Dugong / Nguva (legendary) | Critically endangered; only 2 seen in 2023 aerial survey | Seagrass grazer → simulation, food-web modelling |
| **Humpback whale** | *Megaptera novaeangliae* | Nyangumi | Breeding population in WIO | Song/communication → sound, arrays, frequency |
| **Coral reefs** | Mixed species (400+ in WIO) | Miamba | Climate refuge off Kenya/Tanzania | Ecosystem → objects/classes, interdependence |
| **Seagrass beds** | Mixed species | Mwani wa baharini | Threatened by development | Habitat map → grids, coordinates, arrays |
| **Mangroves** | *Rhizophora mucronata*, *Avicennia marina* | Mikoko | Important nurseries and carbon sinks | Root networks → graph theory, networks |

All species are used as **pedagogical metaphors** and conservation hooks. No learner is required to handle real animals.

## 7. Final Project: Mako Shark Expedition

In weeks 45–52 learners build the capstone: a **Mako Shark Research Dashboard / Mini-Game** in Python (text or Pygame Zero) that must demonstrate:

1. Real data about one Kenyan marine species.
2. User input and output.
3. A loop or list.
4. A conditional decision.
5. A function.
6. A conservation message.
7. Peer presentation.
