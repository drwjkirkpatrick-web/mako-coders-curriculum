# Kenyan Ocean Wildlife Reference

> *Domain reference for the Mako Coders curriculum. Every animal below is a real species documented off the Kenyan coast or in the wider Western Indian Ocean region. Used as coding metaphors and conservation hooks.*

---

## 1. Shortfin Mako Shark — *Isurus oxyrinchus*

| Attribute | Detail |
|-----------|--------|
| Common names | Shortfin mako, blue pointer, bonito shark |
| Swahili | Papa (general shark), mako |
| IUCN status | Vulnerable globally |
| Size | Up to ~3.7 m (12 ft), average 1.8–2.1 m |
| Lifespan | Up to 30 years |
| Reproduction | Live-bearing; litters small (<25 pups every 2–3 years) |
| Habitat | Tropical and temperate waters >16°C; epipelagic to ~500 m; juveniles near coast, adults offshore |
| Kenyan relevance | Found throughout the Indian Ocean, including Kenyan pelagic waters; taken as bycatch and targeted catch |
| Coding metaphor | **Speed, efficiency, real-time decisions** — the mako is one of the fastest sharks and a top predator. Learners model fast data processing, collision detection, and optimising algorithms. |
| Conservation message | Makos mature late and have few pups; overfishing threatens them. Reporting catches helps science. |

## 2. Whale Shark — *Rhincodon typus*

| Attribute | Detail |
|-----------|--------|
| Swahili | Papa mafia |
| IUCN status | Endangered |
| Size | World’s largest fish, up to ~18 m |
| Diet | Filter-feeder on plankton, fish eggs, and small fish |
| Habitat | Coastal and offshore; seasonal visitor to Kenyan waters |
| Coding metaphor | **Filtering big data** — whale sharks sieve huge volumes of water. Learners use loops to filter lists and arrays. |
| Conservation message | Boat strikes and tourism pressure harm whale sharks; responsible ecotourism rules protect them. |

---

## 3. Spinner Dolphin — *Stenella longirostris*

| Attribute | Detail |
|-----------|--------|
| Swahili | Pomboo |
| IUCN status | Varies by population; common in WIO |
| Behaviour | Famous for spinning leaps; travels in pods |
| Coding metaphor | **Rotation and animation** — learners use angles, turns, and repeats to make dolphins spin. |
| Conservation message | Boat noise and harassment disrupt resting pods; keep a respectful distance. |

---

## 4. Indian Ocean Humpback Dolphin — *Sousa plumbea*

| Attribute | Detail |
|-----------|--------|
| Swahili | Pomboo ya Bahari Hindi |
| IUCN status | Critically Endangered in East Africa |
| Habitat | Shallow coastal waters, often near river mouths and mangroves |
| Kenyan context | Kenya’s most dominant yet most vulnerable marine megafauna; only ~1,029 dolphins sighted across whole coast in 2023 survey |
| Coding metaphor | **Rare-event detection / if-else** — learners write code that decides "is this a humpback dolphin sighting?" based on traits. |
| Conservation message | Coastal development and gillnets are major threats; community reporting saves lives. |

---

## 5. Green Sea Turtle — *Chelonia mydas*

| Attribute | Detail |
|-----------|--------|
| Swahili | Kobe ya kijani |
| IUCN status | Endangered |
| Diet | Herbivore as adult; seagrass and algae |
| Nesting | Returns to same beach; lays eggs in cycles |
| Coding metaphor | **State machines and timers** — nesting stages (arrive, dig, lay, cover, return) map to program states. |
| Conservation message | Nesting beaches need darkness and quiet; plastic and poaching are threats. |

## 7. Dugong — *Dugong dugon*

| Attribute | Detail |
|-----------|--------|
| Swahili | Dugong; historically linked to the “nguva” mermaid legend |
| IUCN status | Critically Endangered; possibly functionally extinct in parts of East Africa |
| Diet | Strict seagrass grazer |
| Kenyan context | Groups of up to 500 recorded in the 1960s; only 2 seen in 2023 aerial survey |
| Coding metaphor | **Simulation and food-web modelling** — seagrass consumption vs. regrowth over time. |
| Conservation message | Seagrass beds are disappearing; protecting them protects dugongs, turtles, and fish nurseries. |

## 8. Humpback Whale — *Megaptera novaeangliae*

| Attribute | Detail |
|-----------|--------|
| Swahili | Nyangumi |
| IUCN status | Least Concern globally, but populations recovering |
| Behaviour | Breeding population migrates to WIO; males sing complex songs |

## 9. Coral Reefs — Mixed species (~400+ in WIO)

| Attribute | Detail |
|-----------|--------|
| Swahili | Miamba |
| Kenyan context | Rare climate refuge discovered off Kenya/Tanzania; corals thrive in cool pocket near deep channels from Mt. Kilimanjaro/Usambara runoff |
| Coding metaphor | **Objects/classes and interdependence** — coral polyps, fish, and algae as objects that interact. |
| Conservation message | Reefs support fisheries, tourism, and coastline protection; bleaching and destructive fishing damage them. |

## 10. Seagrass Beds — Mixed species

| Attribute | Detail |
|-----------|--------|
| Swahili | Mwani wa baharini |
| Role | Nursery habitat, carbon store, food for dugongs and green turtles |
| Threats | Salt pans, shrimp farms, sand mining, siltation |
| Coding metaphor | **Grids and coordinates** — map seagrass patches on a 2D grid; model grazing and regrowth. |
| Conservation message | Healthy seagrass means healthy fish stocks and climate resilience. |

## 11. Mangroves — *Rhizophora mucronata*, *Avicennia marina*

| Attribute | Detail |
|-----------|--------|
| Swahili | Mikoko |
| Role | Fish nurseries, coastal protection, carbon sinks |
| Threats | Charcoal, construction, firewood extraction |
| Coding metaphor | **Networks and graph theory** — root systems and nursery connections as nodes and edges. |
| Conservation message | Mangrove restoration is climate-smart and supports local fisheries. |

## 12. How Species Map to Coding Concepts

| Coding concept | Anchor species | Weekly activity example |
|----------------|--------------|-------------------------|
| Sequencing | Green turtle nesting | Order the steps a turtle takes to lay eggs. |
| Variables | Whale shark length | Store and update length/age measurements. |
| Loops | Spinner dolphin spins | Repeat a spin animation 10 times. |
| Conditionals | Humpback dolphin ID | Decide if a sighting is a humpback dolphin based on traits. |
| Functions | Mako shark hunt | Reuse a "chase prey" block/function. |
| Lists/arrays | Whale shark diet | Store plankton types in a list and filter them. |
| Events | Dolphin pod click | Respond to a key press to make dolphins jump. |
| Debugging | Coral bleaching | Find the bug that turns healthy coral white. |
| Coordinates | Seagrass grid | Place seagrass patches at (x, y). |
| Dictionaries | Marine species database | Map species name → status → threat. |
| Classes/objects | Coral reef ecosystem | Model polyps, fish, and algae as objects. |
| Networking/APIs | Mako tracking | Fetch (or simulate) tagged shark location data. |
| Data visualisation | Survey results | Plot 2023 aerial survey sightings. |
| Project cycle | Mako Shark Expedition | Design → Code → Create → Connect → Evaluate. |
